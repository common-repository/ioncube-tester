== Description ==

A simple plugin that helps to determine if the ionCube loaders are installed correctly for your WP site.

If the IonCube Loaders are not installed or working on your server, contact your hosting provider and ask to have them installed.

This plugin need only be run on a single site, if you have multiple sites hosted on the same server.

== Installation ==

1. Go to the dashboard of your WP install and upload the zip file.
2. Click Install then click Activate.
3. Open the Settings tab, then click on IonCube Tester.

== Frequently Asked Questions ==

= What do I do if ionCube loaders are not on my server? =

Contact your hosting provider and ask to have them installed.
